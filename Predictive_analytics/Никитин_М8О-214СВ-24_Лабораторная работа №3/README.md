# Лабораторная работа №3

Проект содержит решения:

1. Загрузка parquet-файлов за 2019–2025 годы.
2. EDA: пропуски, тренды, сезонность, распределения, box-plot, Q-Q plot, корреляции, межгородской анализ, выбросы.
3. Анализ стационарности: ADF-тест и дифференцирование.
4. Feature engineering: календарные, циклические, лаговые, скользящие, динамические и агрегированные признаки.
5. PCA-визуализация разделимости климатических типов.
6. Классификация климатического типа по 30-дневному окну.
7. Оценка классификации: Accuracy, Precision, Recall, F1-score, ROC-AUC, confusion matrix, анализ ошибок, permutation importance.
8. Прогнозирование температуры на 30 дней вперед отдельной Ridge-моделью для каждого города с подбором `alpha` по validation-2024.
9. Оценка прогноза: MAE, RMSE, MAPE_safe, WAPE, direction accuracy, Shapiro-Wilk, Ljung-Box, White test.
10. Интеграция классификации и прогнозирования в единый pipeline: climate-classifier → city-router → city-specific forecasting model.
11. Итоговая сводная таблица и интерпретация результатов.


## Интерпретация результатов

- Confusion matrix показывает, какие климатические типы модель путает чаще всего.
- ROC-AUC One-vs-Rest показывает качество многоклассовой классификации.
- MAE и RMSE оценивают среднюю ошибку прогноза температуры.
- MAPE_safe считается только для точек, где |T| >= 1°C, потому что обычный MAPE некорректен около 0°C и при отрицательных температурах. Основная относительная метрика — WAPE.
- Direction accuracy показывает, насколько модель угадывает направление изменения температуры.
- Shapiro-Wilk, Ljung-Box и White test используются для диагностики остатков.
